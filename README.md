ТГ бот на aiogram, решил заспидранить новую библиотеку, получилось за 3 часа, 1 минуту и 5 секунд (скорострел).

Бот наподобие green-bot в дискорде, короче ты ему файлики с музыкой скидываешь, и потом по кнопочке их получить можешь
в виде голосового сообщения.

Мой первый проект из больше чем 1 файла, решил выложить т.к. по документации хуй поймёшь что от тебя требуется,
может кому-то поможет (даже закомментить попытался).

Впринципе всё что требуется чтобы запустить себе этого бота, это строчка TOKEN="с твоим токеном" в .env, вроде.
